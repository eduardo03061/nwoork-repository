# Front-nwoork

Esta Aplicacion fue realizada en VueJs

## Requisitos

```sh
    Node -v > 14.8
```

Primeros pasos

```sh
npm install
```

Comandos para ejecutar:

```sh
npm run dev
```

Variables de entorno:

```sh
VITE_BACKEND_URL=http://localhost:3000
```
